# StarPivot 项目依赖引用梳理

## 一、项目结构总览

```
StarPivot (根)
├── star-pivot-dependencies   # BOM：统一版本管理，无代码
├── star-pivot-framework      # 框架层聚合（仅聚合，无直接依赖）
│   ├── star-pivot-framework-core   # 底座：domain/异常/常量/工具等（包名 com.star.pivot.framework）
│   ├── star-pivot-framework-log
│   ├── star-pivot-framework-file
│   ├── star-pivot-framework-web
│   ├── star-pivot-security
│   └── star-pivot-framework-boot-starter   # 应用层一站引入：security + core
├── star-pivot-module         # 业务模块聚合
│   ├── star-pivot-system
│   ├── star-pivot-generator
│   └── star-pivot-quartz
├── star-pivot-controller     # 应用入口（可执行 JAR）
└── star-pivot-ui             # 前端（Vue3 + Vite，独立依赖体系）
```

**分层原则**：dependencies → framework(core/log/file/web/security) → module(system/generator/quartz) → controller，无跨层反向依赖。通用能力由 framework-core 提供，不再单独存在 star-pivot-common 模块。

---

## 二、根 POM 与版本管理

- **根 POM**：`StarPivot/pom.xml`
  - 通过 `<dependencyManagement>` 引入 `star-pivot-dependencies`（BOM），**不直接声明依赖**。
  - 子模块版本由 BOM 统一管理，子模块中一般不写 `<version>`。

- **star-pivot-dependencies**
  - 仅做版本管理（`<dependencyManagement>` + `<properties>`）。
  - 管理：Spring Boot 3.2.10、MyBatis-Plus、MySQL、Druid、JWT、Hutool、POI/EasyExcel、MinIO、Quartz、SpringDoc 等第三方版本。
  - 同时管理所有**内部模块**的 artifact 与版本（如 `star-pivot-framework-core`、`star-pivot-security`、`star-pivot-system` 等）。

## 四、各模块依赖明细

### 4.1 star-pivot-dependencies（BOM）

- **职责**：仅 `<dependencyManagement>` + `<properties>`，无 `<dependencies>`。
- **不参与构建**：无源码，仅被根 POM import。

---

### 4.2 star-pivot-framework-core

- **职责**：框架底座，通用 domain、异常、常量、SQL 工具、基础 utils。
- **内部依赖**：无。
- **第三方依赖**：
  - spring-context-support, spring-boot-starter-validation, spring-boot-starter-security
  - commons-lang3, jackson-databind, lombok
  - jjwt-api, hutool-all, fastjson2, easyexcel

---

### 4.3 star-pivot-framework-log

- **职责**：操作日志（@Log、LogUtils、IP/UA/脱敏等）。
- **内部依赖**：`star-pivot-framework-core`。
- **第三方**：spring-web, jakarta.servlet-api, jackson-databind, yauaa。

---

### 4.4 star-pivot-framework-file

- **职责**：文件存储（MinIO、阿里云 OSS）。
- **内部依赖**：`star-pivot-framework-core`。
- **第三方**：spring-context, spring-web, commons-io, aliyun-sdk-oss, minio, lombok。

---

### 4.5 star-pivot-framework-web

- **职责**：Web 层抽象（如登录用户信息等）。
- **内部依赖**：`star-pivot-framework-core`。
- **第三方**：无额外第三方（仅 core 传递）。

---

### 4.6 star-pivot-security（groupId: com.star.pivot）

- **职责**：JWT、Security 配置、黑名单、放行路径等。
- **内部依赖**：`star-pivot-framework-core`、`star-pivot-framework-log`、`star-pivot-framework-file`、`star-pivot-framework-web`。
- **第三方**：
  - spring-boot-starter-security, spring-boot-starter-web
  - spring-boot-starter-data-redis
  - spring-boot-starter-validation
  - jjwt-api, jjwt-impl, jjwt-jackson
  - lombok, spring-boot-configuration-processor(optional)

---

### 4.7 star-pivot-framework-boot-starter

- **职责**：应用层“一站引入”框架能力（security + core）。
- **内部依赖**：
  - star-pivot-security（传递 core / log / file / web）
  - star-pivot-framework-core
- **第三方**：无（全部传递）。

---

### 4.8 star-pivot-system（groupId: com.star.pivot）

- **职责**：系统管理（用户、角色、菜单、部门、字典、通知、在线用户、监控等）。
- **内部依赖**：
  - star-pivot-framework-core
  - star-pivot-security
- **第三方**：
  - spring-boot-starter-aop, commons-lang3
  - mybatis-plus-boot-starter
  - aliyun-sdk-oss, druid-spring-boot-3-starter
  - oshi-core

---

### 4.9 star-pivot-generator（groupId: com.star.pivot）

- **职责**：代码生成（Velocity 模板、表/列配置）。
- **内部依赖**：
  - star-pivot-framework-core
  - **star-pivot-system**（读取表结构、字典等）
- **第三方**：
  - mybatis-plus-boot-starter, mybatis-plus-generator
  - velocity-engine-core
  - druid-spring-boot-3-starter
  - commons-io, lombok

---

### 4.10 star-pivot-quartz（groupId: com.star.pivot）

- **职责**：定时任务（Quartz）。
- **内部依赖**：
  - star-pivot-framework-core
  - star-pivot-security
  - star-pivot-system（如清理操作日志任务等）
- **第三方**：
  - spring-boot-starter, spring-boot-starter-quartz
  - mybatis-plus-boot-starter
  - commons-lang3, lombok(provided)

---

### 4.11 star-pivot-controller（应用入口）

- **职责**：Spring Boot 启动、Web、配置、Controller 聚合。
- **内部依赖**：
  - star-pivot-framework-boot-starter（已聚合 security + core + file，应用层无需再显式依赖 file）
  - star-pivot-system
  - star-pivot-generator
  - star-pivot-quartz
- **第三方**：
  - spring-boot-starter-web, spring-boot-devtools(optional)
  - druid-spring-boot-3-starter, spring-boot-starter-security
  - mysql-connector-j
  - lombok, hutool-all, commons-lang3
  - mybatis-plus-boot-starter
  - spring-boot-starter-actuator, oshi-core
  - springdoc-openapi-starter-webmvc-ui

## 六、前端 star-pivot-ui 依赖（独立体系）

- **技术栈**：Vue 3、Vite 7、Element Plus、Pinia、Vue Router、TypeScript。
- **依赖管理**：`package.json`（pnpm，Node >= 20.19.0）。
- **与后端关系**：通过 HTTP 调用 controller 暴露的 API，无 Maven 依赖关系。

主要运行时依赖包括：vue、vue-router、pinia、element-plus、axios、echarts、dayjs、xlsx、@wangeditor/editor 等；构建与规范：vite、typescript、sass、eslint、prettier、stylelint 等。

---

## 七、构建与使用说明

1. **首次或清理后构建**：需先安装 BOM，再整体构建。
   ```bash
   mvn install -f star-pivot-dependencies/pom.xml
   mvn clean install -DskipTests
   ```
2. **仅运行应用**：
   ```bash
   mvn spring-boot:run -pl star-pivot-controller -am
   ```
3. **依赖版本**：以 `star-pivot-dependencies/pom.xml` 中 `<properties>` 与 `<dependencyManagement>` 为准，子模块一般不重复写版本。

## 八、依赖流向小结

| 层级     | 模块 | 依赖的上层 |
|----------|------|------------|
| BOM      | star-pivot-dependencies | 无（仅被根 import） |
| 底座     | framework-core | 无内部模块 |
| 框架能力 | framework-log / file / web | framework-core |
| 安全     | star-pivot-security | core + log + file + web |
| 启动聚合 | framework-boot-starter | security + core |
| 业务     | star-pivot-system | framework-core + security |
| 业务     | star-pivot-generator | framework-core + star-pivot-system |
| 业务     | star-pivot-quartz | framework-core + security + system |
| 应用     | star-pivot-controller | boot-starter + file + system + generator + quartz |

以上即为整个 StarPivot 项目的依赖引用梳理。
